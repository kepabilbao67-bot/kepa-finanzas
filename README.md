# Kepa Tecnología — Sitio web

Landing page responsiva de empresa de tecnología, con una sección de **custodia segura de oro y plata** ilustrada con gráficos vectoriales (SVG).

## Estructura

```
index.html          Sitio principal (hero, servicios, custodia, nosotros, proyectos, contacto)
styles.css          Estilos (responsivo, animaciones)
script.js           Interactividad (menú móvil, scroll reveal, contadores, validación)
galeria.html        Galería para ver y descargar las ilustraciones
img/
 ├─ boveda-custodiada.svg        Caja fuerte con guardia, cámaras y escudo
 ├─ boveda-oro-plata.svg         Caja fuerte con lingotes
 ├─ lingotes-oro-plata.svg       Primer plano de lingotes (oro 9999 / plata 999)
 └─ banner-boveda-seguridad.svg  Banner ancho 1280x420
```

## Uso

Abre `index.html` en cualquier navegador. No requiere dependencias ni compilación.

## Tecnologías

- HTML5
- CSS3 (Grid, Flexbox, gradientes, animaciones)
- JavaScript (vanilla, sin librerías)
- Ilustraciones SVG vectoriales
